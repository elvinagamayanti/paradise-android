cara run
1. buat database baru
2. ubah application.properties
3. run app
4. masuk ke phpmyadmin
5. insert data role_user dan role_admin ke tabel roles
6. register akun
misal :
{
    "name": "Ada Deh",
    "email": "yyy@stis.ac.id",
    "password": "ppkuas2023"
}
7. login akun
{
    "email": "yyy@stis.ac.id",
    "password": "ppkuas2023"
}
8. tinggal ganti proses bisnis (entity, service, controller, repository, dll)

*NB : ganti ROLE dilakukan di Tabel user_roles (Cek PHPMyAdmin)
